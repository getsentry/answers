## The Problem

How can I concatenate string variables in Bash?

## The Solution

You can use the [substitution syntax](https://tldp.org/LDP/Bash-Beginners-Guide/html/sect_10_03.html) (`${}`) as follows:

```sh
h="Hello"
w="world"
hw="${h} ${w}"
echo $hw # will output "Hello world"
```
